YouTube Web Scraper Project:-

Project Overview:-

This project is a Python-based web scraper designed to extract information from YouTube. Using this tool, you can automate the process of gathering data such as video titles, descriptions, views, likes, comments, and more. The project is built using a Jupyter notebook, allowing for interactive data collection and analysis.

Features:-

Automated Data Extraction: Scrape video details like title, description, views, likes, and comments.
Interactive Analysis: Use Jupyter notebook to interactively run and modify the scraping process.
Customizable Parameters: Adjust scraping parameters to target specific videos or channels.
